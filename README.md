<h1 align="center">Buttons</h1>
<h4 align="center">Telegram bot to add buttons to a conversation</h4>

## Идея
Установке любых кнопок в любую беседу, по запросам покупателя.

Бот принимает заявку, отправляет ее мне, а далее я редактирую код для добавления конкретной клавиатуры, захожу в беседу и активирую ее по команде боту.
